# Squinks Arcade

**▶ Play it live: https://sleeprived.github.io/squinks-arcade/**

An installable, offline arcade hub of nineteen browser games. Open the link, pick
a game, play. **Add to Home Screen** to install it like an app; it works offline
after the first load, and scores and chip bankrolls save per-device.

## Games
Chess (vs Stockfish), Snake, 2048, Tetris, Minesweeper, Connect 4 (vs AI), Doodle
Jump, Blackjack, Video Poker, Muncher, Star Divers, Twin Talon, Breakout,
Asteroids, 15-Puzzle, Reversi (vs AI), Simon, Whack-a-Mole, and Roulette —
grouped by category (Arcade / Puzzle / Skill / Cards) with a filter bar.

## Privacy
Runs entirely in your browser — no accounts, no servers, no network calls.
Everything is saved on your device.
